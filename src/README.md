# Lowe Automation Premium Website

A complete premium replacement website for Lowe Automation.

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Build check

```bash
npm run build
```

## Included

- Premium animated homepage
- Services page
- Projects/case studies page
- About page
- Contact page with professional form UI
- SEO metadata
- Sitemap
- Robots.txt
- Reusable components
- Responsive navigation
- Dark premium software-company design
